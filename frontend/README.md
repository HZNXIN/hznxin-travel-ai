# Event Horizon - 前端使用说明

## 🚀 快速启动

### 方法1：直接双击打开（最简单）
```
双击 index.html 文件
即可在默认浏览器中打开
```

### 方法2：本地服务器（推荐）
```bash
# 进入前端目录
cd frontend

# Python 3
python -m http.server 8000

# 或者 Python 2
python -m SimpleHTTPServer 8000

# 访问
http://localhost:8000
```

## 🧪 测试场景

### 场景1：晴天下午游览
```
1. 时间滑块 → 14:00
2. 天气按钮 → ☀️ 晴
3. 观察：拙政园评分提升（户外加分）
4. 点击"拙政园"节点 → 查看详情
```

**预期结果**：
- 拙政园：0.96分 (1.20x天气加分)
- 苏州博物馆：0.80分 (晴天正常)

### 场景2：雨天推荐（核心演示）⭐
```
1. 时间滑块 → 14:00
2. 天气按钮 → 🌧️ 小雨
3. 观察：博物馆跃升第一（室内加分）
4. 点击"苏州博物馆"节点 → 查看AI推荐理由
```

**预期结果**：
- 苏州博物馆：0.96分 (1.20x室内加分) ⭐
- 拙政园：0.56分 (0.70x户外降分)
- AI理由："雨天参观更佳，是当前场景的最优选择"

### 场景3：晚高峰拥堵熔断（核心演示）⭐⭐
```
1. 时间滑块 → 18:00
2. 观察左侧阻尼因子变化：
   - L_zone: 0.30 (CBD严重拥堵)
   - 综合修正: 0.33x 🔴
3. 观察画布：金鸡湖CBD边变红色虚线
4. 点击"金鸡湖CBD"节点
5. 查看拥堵熔断警告
```

**预期结果**：
- 金鸡湖CBD：0.27分 (0.33x拥堵熔断)
- 红色虚线边
- 警告："晚高峰严重拥堵，系统已触发自动熔断"

### 场景4：暴雨极端天气
```
1. 天气按钮 → ⛈️ 暴雨
2. 观察：户外景点大幅降权
```

**预期结果**：
- 拙政园：0.40分 (0.50x极端天气)

## 📊 技术实现

### 技术栈
- **Vue 3** - 响应式框架
- **ECharts 5** - 图表可视化
- **Tailwind CSS** - 快速样式
- **纯前端** - 零配置，即开即用

### 数据流
```
用户交互 → Vue响应式更新 → 计算阻尼因子 → 更新ECharts → 展示结果
```

## 操作流程（照着做就行）

### 第1步：启动任务
```
👆 点击左下角：🚀 START MISSION
```
**看到什么**：
- ✅ "Session created: session_0"
- 🗺️ 地图显示苏州站（绿色起点标记）
- 💬 AI显示："➡️ NEXT STEP"

---

### 第2步：获取推荐
```
👆 点击左下角：➡️ GET OPTIONS
```
**看到什么**（等2-3秒）：
- 🗺️ 地图出现9个数字标记（1, 2, 3...）
- 📋 右侧时间轴出现9个选项
- 📦 右下角出现3张推荐卡片
- 💬 AI显示："💡 PLEASE SELECT"

**现在你可以：**
- 在地图上看到所有候选点的位置
- 时间轴显示每个点的详细信息

---

### 第3步：查看详情（必做！）
```
👆 点击右侧时间轴的任意一项
   或
👆 点击右下角推荐卡片
```

**示例：点击时间轴 #1 苏州博物馆**

**看到什么**：
- ✅ 该项高亮（青色边框）
- 📊 左上雷达图更新（显示5维张力）
- 💭 AI核心更新："🎯 SELECTED #1"
- 💬 显示AI解释："我觉得可以去...虽然不知道是不是会太正式"
- 🗺️ 地图自动聚焦到该地点

**💡 提示**：可以多次点击不同选项对比！

---

### 第4步：确认前往
```
👆 点击左下角：✅ CONFIRM
```
**发生什么**：
- 📍 前往苏州博物馆
- 💰 扣除费用（如 ¥47）
- ⏰ 增加时间（如 1.9小时）
- 🗺️ 地图标记移动到新位置
- 📝 日志显示："Traveled to: 苏州博物馆"

---

### 第5步：自动继续
```
🔄 系统自动执行（1秒后）
```
**发生什么**：
- 自动获取下一步推荐
- 回到第2步
- 继续规划下一个景点

**重复第2-5步，完成3天行程！**

## ⚠️ 常见疑问

### ❓ Q: 不知道点哪里？
**A**: 按顺序点：
1. START MISSION（左下）
2. GET OPTIONS（左下）
3. 时间轴任意项（右侧）← **必须点！**
4. CONFIRM（左下）

### ❓ Q: 地图没有标记？
**A**: 
- 点击 GET OPTIONS 后才会显示
- 刷新页面重试

### ❓ Q: 点了时间轴没反应？
**A**: 
- 看雷达图是否更新（左上）
- 看AI解释是否变化（中间）
- 看地图是否聚焦（中间）
- 如果都更新了，就是成功了！

### ❓ Q: CONFIRM后发生了什么？
**A**: 
- 看终端日志（左侧）
- 看地图标记（中间）
- 1秒后自动显示新推荐

## 🎯 重点提示

### ✅ 必须点击的3个按钮

```
1️⃣ 🚀 START MISSION  ← 开始
2️⃣ ➡️ GET OPTIONS     ← 获取推荐
3️⃣ ✅ CONFIRM         ← 确认前往
```

### 👆 必须点击查看详情

```
获取推荐后 → 必须点击时间轴或推荐卡片
→ 查看雷达图、AI解释
→ 然后才能CONFIRM
```

### 🗺️ 地图显示逻辑

```
START后        → 显示起点（1个标记）
GET OPTIONS后  → 显示所有候选（9个标记）
点击选项后     → 聚焦到该点（放大）
CONFIRM后      → 移动到新位置
```

## 💡 额外技巧

### 对比多个选项
